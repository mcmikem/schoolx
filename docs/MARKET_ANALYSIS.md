# SkoolMate OS — Market Analysis & Pricing Strategy

## Which Plan for Which School

### Rural Primary · under 200 students

**Starter** (UGX 2,000/student)

- Needs basics, limited budget, offline critical

### Urban Primary · 200–500 students

**Growth** (UGX 3,500/student)

- Needs SMS, parent portal, fuller finance

### Secondary School · any size

**Growth → Enterprise**

- Start Growth, upsell on UNEB + payroll

### Large School · 500+ students

**Enterprise** (UGX 5,500/student)

- Needs payroll, audit, AI, full compliance

### Diocese / School Group / NGO

**Lifetime / white-label**

- Wants ownership across multiple schools

---

## Unique Differentiators (Only SkoolMate Has)

1. **NCDC syllabus + scheme of work** - No competitor has full NCDC curriculum alignment built in. This is a genuine differentiator for secondary schools trying to track coverage.

2. **MTN MoMo + Airtel Money integration** - Native mobile money payment collection is huge for rural Uganda where parents don't use banks. Nobody else has this built in properly.

3. **Student ID card generation** - Small feature but schools love it — saves them money and admin time. No competitor offers this.

4. **Transport route tracking** - No other Uganda-specific SMS has this. Boarding and semi-urban schools with buses will notice.

5. **DNA + trend analysis** - Deep academic needs analysis is entirely unique. ShuleKeeper has general AI insights but nothing this structured.

6. **White-label + source code license** - Opens a B2B channel — NGOs, school groups, and diocesan networks can buy a branded version. No one else offers this.

## Critical Gaps to Close

### Fix urgently

- **Digital signatures on report cards** - ShuleKeeper just launched this. Small feature but highly visible. Add it fast.
- **Offline mode too deep in tiers** - For rural Uganda that's a dealbreaker. Move basic offline to Growth.

### Close soon

- **UNEB candidate registration** - Fully nail UNEB candidate registration + MoES exports to win every secondary school conversation.
