# UNEB Registration & Report Card Requirements

## UNEB e-Registration Requirements (2025/2026)

### Registration Portal

- **URL**: ereg.uneb.ac.ug
- **Method**: Windows desktop app / Web portal / Bulk upload

### Required Fields for Each Candidate

| Field               | Description                    | Format              |
| ------------------- | ------------------------------ | ------------------- |
| Full Name           | As it appears in birth record  | Text (max 50 chars) |
| Date of Birth       | In UNEB format                 | DD/MM/YYYY          |
| Gender              | Male/Female                    | M/F                 |
| Subject Choices     | Optional subjects per exam     | Select from list    |
| Institution Choices | School preferences for A-level | Priority 1, 2, 3    |
| Photograph          | Digital - passport size        | JPG, max 100KB      |
| Index Number        | Auto-generated by UNEB         | -                   |
| Centre Number       | School's UNEB center number    | Assigned by UNEB    |

### Exam Fees (2025 Rates)

| Exam | Normal Fee  | Late Fee (50% surcharge) |
| ---- | ----------- | ------------------------ |
| PLE  | UGX 34,000  | UGX 51,000               |
| UCE  | UGX 164,000 | UGX 246,000              |
| UACE | UGX 186,000 | UGX 279,000              |

### Key Requirements for New Curriculum (LSC)

For S4 (UCE candidates):

1. **Continuous Assessment (CA)** scores from S3 and S4 terms
2. **Activities of Integration (AOI)** scores per subject per term
3. **Project marks** for each subject
4. **Subject Observation Checklist** scores
5. **CAI (Continuous Assessment Items)** for at least 2-3 levels

### Registration Status Flow

```
pending → registered → confirmed → (exam) → results
```

## NCDC Report Card Requirements (New Curriculum)

### Assessment Structure (Competency-Based)

| Component                 | Weight | Description              |
| ------------------------- | ------ | ------------------------ |
| Summative Exam            | 80%    | End of term exam         |
| Activities of Integration | 20%    | Chapter-based activities |

### Report Sections Required

1. **Header**
   - School name, logo, address
   - Term, Academic Year
   - Student Level/Class

2. **Student Information**
   - Name, ADM No., Stream
   - Date of birth, Gender
3. **Subject Scores Table**
   - Subject name
   - CA1, CA2, CA3, CA4 scores
   - Project score
   - AOI (Activities of Integration) score
   - Total marks
   - Grade
   - Competency comment

4. **Generic Skills** (optional - can be combined with subject comments)
   - Critical thinking
   - Communication
   - Collaboration
   - Creativity

5. **Summary**
   - Total marks/Average
   - Division (for O-level)
   - Points (for A-level)
   - Position in class

6. **Comments**
   - Subject Teacher Comment
   - Class Teacher Comment
   - Headteacher Comment

7. **Signatures**
   - Subject teachers (optional - with digital signatures)
   - Class teacher
   - Headteacher (with digital signature)

### Grading Scale (New Curriculum)

**O-Level (S1-S4)**:
| Grade | Marks Range | Descriptor |
|-------|-------------|-------------|
| D1 | 80-100 | Excellent |
| D2 | 75-79 | Very Good |
| C3 | 70-74 | Good |
| C4 | 65-69 | Fairly Good |
| C5 | 60-64 | Satisfactory |
| C6 | 55-59 | Adequate |
| P7 | 50-54 | Pass |
| F8 | 40-49 | Below Expectation |
| F9 | 0-39 | Fail |

**A-Level (S5-S6)**:
| Grade | Points | Descriptor |
|-------|--------|-------------|
| A | 6 | Excellent |
| B | 5 | Very Good |
| C | 4 | Good |
| D | 3 | Satisfactory |
| E | 2 | Pass |
| F | 0-1 | Fail |

### Competency Tracking (Per Subject)

For each subject, track:

- Knowledge outcomes
- Skill outcomes
- Attitude/Values outcomes
